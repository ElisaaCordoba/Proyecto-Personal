README
ABRE SOLO INDEX.TXT (Paginaprincipal.html es el antiguo, le he cambiado de nombre a index.html y tiene ya añadido el jquery dentro del archivo)
Este proyecto personal es una página web diseñada para mostrar información sobre mí, mis intereses y proyectos en los que he trabajado y el contacto. La página está construida con HTML, CSS y Bootstrap para un diseño responsivo y fácil de usar.



Características
Secciones principales: Sobre mí, Mis Proyectos, Contacto.
Buscador: Puedes buscar proyectos específicos en la sección "Mis Proyectos".
(buscas un proyecto y lo tienes que escribir como sale en la página, y te lleva a el, luego debes de borrar en el buscador lo escrito anteriormente para que te salgan los demás)
Desplazamiento suave: La página se desplaza suavemente a la sección de proyectos cuando se realiza una búsqueda.

Tecnologías utilizadas:
HTML5: Para la estructura del contenido.
CSS3: Para estilos personalizados y diseño visual.
Bootstrap: Para un diseño responsivo y componentes listos para usar.
JavaScript/jQuery: Para la funcionalidad de búsqueda y desplazamiento automático
